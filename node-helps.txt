Node se instala por el enlace https://nodejs.org/en/download/package-manager
Si queremos manejar diferentes versiones en Windows es necesario instalar antes NVM(Node Version Manager) https://github.com/coreybutler/nvm-windows?tab=readme-ov-file

Con NMV podemos en vez de instalar node a travez de comandos con nvm:
    nvm install lts -> instalar la versionde node LTS
    nvm list -> lista de versiones de node instaladas
    nvm current -> version de node activa
    nvm use version -> activar version de node 
    nvm -> para saber mas de los comandos abituales de NVM
